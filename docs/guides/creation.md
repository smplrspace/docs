# Creating spaces

The creation (or shall we call it _digitalisation_) of floor plans in Smplrspace is done in the editor. This can currently be done by logging in [the app](https://app.smplrspace.com). You can also embed the editor directly in your own app — refer to the [embedded editor (beta)](/api-reference/editor/overview) page to learn more.

The editor is purpose-built for digitising existing floor plans — tracing walls, doors, windows, and furniture from a reference image. It is not a space planning or CAD tool. Built with minimalism in mind and reusing familiar UX patterns from tools like Powerpoint, our intention is for it to become the easiest floor plan digitisation tool for untrained consumer users.

The editor includes two modes: 2D and 3D. The 2D mode is suitable to add walls, doors, windows, annotations to the space, while the 3D mode is dedicated to furnishing the space. Below are previews of each mode.
